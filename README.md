# Bua Lagbe

**Bua Lagbe** is a comprehensive, premium management platform designed to bridge the gap between households and domestic helpers. Built with the latest web technologies, it offers a seamless, secure, and intuitive experience for both users and administrators.

🔗 **Live Demo:** [bualagbe.vercel.app](https://bualagbe.vercel.app)

---

## ✨ Key Features

### 🔐 Advanced Authentication
- **Google OAuth Integration**: Secure and fast login via Google.
- **Role-based Access Control**: Distinct interfaces and permissions for Users and Admins.
- **Secure Sessions**: Persistent and secure session management.

### 📊 Powerful Admin Dashboard
- **User Insights**: Monitor and manage all registered users and helpers.
- **System Overview**: High-level statistics on platform usage and financials.
- **Resource Management**: Tools to approve, edit, or remove helper profiles.

### 📅 Smart Attendance Tracking
- **Automated Logging**: Track daily attendance for helpers with ease.
- **History Logs**: Review past attendance records to ensure transparency.
- **Status Updates**: Real-time status indicators for current work instances.

### 💰 Integrated Financial Suite
- **Transaction History**: Detailed logs of all payments and financial movements.
- **Wallet System**: Digital wallet for users and helpers to manage balances.
- **Financial Analytics**: Visualize spending and earnings over time.

### 👤 Detailed Helper Profiles
- **Rich Information**: Experience, skills, and ratings at a glance.
- **Profile Customization**: Helpers can maintain professional profiles to showcase their reliability.

## 🚀 Technology Stack

- **Framework**: [Next.js 15+](https://nextjs.org/) (App Router, Server Actions)
- **Library**: [React 19](https://react.dev/)
- **Database**: [MongoDB](https://www.mongodb.com/) with [Mongoose](https://mongoosejs.com/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) (Modern, utility-first design)
- **Authentication**: [Google OAuth](https://developers.google.com/identity/gsi/web/guides/overview) & [JWT](https://jwt.io/)
- **State Management**: React Context & Hooks
- **Deployment**: [Vercel](https://vercel.com/)

## 📁 Repository Structure

```text
├── app/              # Next.js App Router (Pages & APIs)
│   ├── admin/        # Admin Dashboard interface
│   ├── attendance/   # Attendance tracking system
│   ├── financials/   # Financial management
│   ├── profile/      # User profile features
│   └── wallet/       # Digital wallet implementation
├── components/       # Reusable UI components
├── lib/              # Core utilities & DB connections
├── models/           # Mongoose schemas (User, Helper, Transaction)
├── public/           # Static assets (images, icons)
└── package.json      # Dependencies and scripts
```
